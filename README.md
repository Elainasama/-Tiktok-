### 实现功能
|  数据库表   | 对应接口功能  |
|  ----  | ----  |
| User  |  用户注册、用户登录、获取用户信息|
| Video  | 上传视频、获取上传列表、推送视频流 |
| Favorite  | 点赞 、 取消点赞 、 点赞列表 |
| Comment  | 评论 、 删除评论 、 评论列表 |
| Relation  | 关注 、 取消关注 、 关注列表 、 粉丝列表 、 互关列表|
| Message | 发送消息 、 聊天记录 |
